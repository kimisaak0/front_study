HTML

<html> //html 문서 영역을 시작한다.
<head> //문서 정보를 담고 있는 머리글 영역을 시작한다.
<title> //웹브라우저에게 문서의 제목을 알려준다.
<body> //문서의 본문 영역으로, 사용자에게 보이게 될 부분의 시작이다.
<h1> //본문 영역의 제목을 나타낸다. h1~h6 까지 단계를 나눌 수 있다.
<p> //글의 덩어리인 단락을 나타낸다.

<meta charset="utf-8"> //이 문서의 인코딩 설정을 UTF-8로 설정한다.
인코딩? UTF-8?
meta요소 //문서의 정보를 웹 브라우저에게 알려주는 역할을 한다.
empty요소 //내용을 갖지 않는 요소를 의미한다.
</DOCTYPE> //identifier 식별자.

<ul> //비순차 목록 영역을 시작한다.
<ol> //순차 목록 영역을 시작한다.
<li> //목록에서 각각의 데이터를 담는다.

HTML5 outliner
HTML5에서부터는 <section>이라는 문서의 구조를 나누는 의미 요소가 추가되었다.
섹션은 제목을 하나씩만 가질 수 있으며, 제목이 2개 이상 있을 경우 제목의 단계에 따라 섹션이 분리되거나, 하위 섹션이 생성된다.
섹션 요소의 종류는 다음과 같다.
<section> //다용도로 쓸 수 있는 섹션 요소.
<article> //독립적인 문서를 나타날 때 사용하는 섹션 요소. 블로그 포스트 등에 쓴다.
<nav> //내비게이션을 나타낼 때 사용하는 섹션 요소.
<aside> //문서의 주요 내용과 분리되는 보조 내용(광고, 즐겨찾기, 링크 등)을 나타낼 때 사용하는 섹션요소
